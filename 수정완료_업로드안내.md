## 🚀 GitHub 업로드 방법

### 1. 기존 파일 교체
1. **https://github.com/ape-boy/replit** 접속
2. **기존 `index.html` 파일 클릭**
3. **편집(연필) 아이콘** 클릭
4. **전체 내용을 첨부파일 내용으로 교체**
5. **"Commit changes"** 클릭

### 2. 새로 업로드
1. **기존 `index.html` 삭제**
2. **"Upload files"** 클릭
3. **첨부파일을 `index.html`로 업로드**
